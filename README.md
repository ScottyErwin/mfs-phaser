# Monster Farm Smash - Phaser 3 Version

A cozy monster farming + smash combat game built with Phaser 3.

## How to Run

1. Open `index.html` in a modern browser (Chrome/Firefox recommended)
2. Use Arrow Keys or WASD to move the player

## Project Structure

- `index.html` - Main entry point
- `js/config.js` - Phaser game configuration
- `js/main.js` - Game initialization
- `js/scenes/` - Game scenes (BootScene, FarmScene, etc.)
- `js/entities/` - Player, Monster, and other game objects
